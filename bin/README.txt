  youtube_downloader-1.0-debug.apk

  • Підписано автоматично
  • Швидка компіляція

RELEASE версія (для розповсюдження):
  youtube_downloader-1.0-release.apk (скоро)
  • Розмір: ~50-70 MB
  • Для Google Play
  • Вручну підписана
  • Оптимізована


ВСТАНОВЛЕННЯ:
─────────────

1. USB кабель:
   adb install youtube_downloader-1.0-debug.apk

2. Файл на телефон:
   • Скопіюйте .apk
   • Відкрийте файл менеджер
   • Натисніть на файл
   • Дозвольте встановлення

3. Беспроводное (ADB over TCP):
   adb connect 192.168.1.100:5555
   adb install youtube_downloader-1.0-debug.apk

  • READ_EXTERNAL_STORAGE - для доступу до файлів


РОЗВ'ЯЗАННЯ ПРОБЛЕМ:
─────────────────────

"Cannot install"
  → Увімкніть встановлення з невідомих джерел
    Settings > Security > Unknown sources

"App crashes"
  → Перевстановіть APK
  → Видаліть дані: Settings > Apps > YouTube Downloader > Clear

"No download permission"
  → Дозвольте доступ до файлів
    Settings > Apps > YouTube Downloader > Permissions
