🔧 НАЛАШТУВАННЯ (buildozer.spec)

Версія:
  version = 1.0

Назва пакету:
  package.name = youtube_downloader
  package.domain = com.vladix

Дозволи:
  android.permissions = INTERNET,WRITE_EXTERNAL_STORAGE,READ_EXTERNAL_STORAGE

Мінімальна Android:
  android.minapi = 21  (Android 5.0+)


❌ ЯКЩО ВИНИКЛИ ПРОБЛЕМИ

"Android SDK not found"
  → Встановіть ANDROID_SDK_ROOT змінну
  → export ANDROID_SDK_ROOT=~/Android

"Java not found"
  → sudo apt-get install openjdk-11-jdk

"Buildozer not found"
  → pip install buildozer

"yt-dlp not available"
  → Додайте в requirements buildozer.spec

"FFmpeg not available"
  → На Android буде вбудований FFmpeg через p4a

Повільна компіляція
  → НОРМАЛЬНО! Перша компіляція займає 15-30 хвилин
  → Наступні буде швидше (~5-10 хвилин)
