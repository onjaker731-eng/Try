📋 ФАЙЛИ ПРОЕКТУ:

Основний скрипт:
  📄 ForTheDownloader.py      - Основна програма Python

Виконувальні файли:
  🖥️  dist/YouTube-Downloader  - Linux/macOS виконуваний файл
  🖥️  dist/YouTube-Downloader.exe - Windows/Linux (копія)
  🔨 run.sh                   - Скрипт запуску для Linux/macOS

Для Windows:
  📝 build_exe.bat            - Автоматична компіляція .exe на Windows
  📖 BUILD_EXE.md             - Повні інструкції по компіляції

Документація:
  📖 README.md                - Повна документація
  💡 QUICKSTART.md            - Швидкий старт
  💡 TIPS.md                  - Поради та приклади
  📝 EXE_INFO.txt             - Інформація про файли

Конфігурація:
  📋 requirements.txt         - Python залежності
  ⚙️  build.spec              - PyInstaller конфігурація


🚀 ШВИДКИЙ ЗАПУСК:

Для Linux/macOS:
───────────────
chmod +x run.sh
./run.sh

Для Windows (якщо .exe компільований):
──────────────────────────────────────
dist\YouTube-Downloader.exe

Як Python скрипт (будь-де):
─────────────────────────────
source .venv/bin/activate    # На Windows: .venv\Scripts\activate
python ForTheDownloader.py

📲 ЯК ЗАВАНТАЖУВАТИ ВІДЕО:

1. Запустіть програму
2. Виберіть опцію з меню (1-3)
3. Введіть посилання на YouTube
4. Файл буде збережено в папці 'downloads/'

Приклади посилань:
  • https://www.youtube.com/watch?v=VIDEO_ID
  • https://www.youtube.com/shorts/VIDEO_ID

💾 .EXE ФАЙЛ - ВАЖЛИВО:

На Linux машині PyInstaller створює Linux ELF файл.
Справжній Windows .exe потребує компіляції на Windows!

Якщо потрібен Windows .exe:
  1. Скопіюйте проект на Windows
  2. Запустіть: build_exe.bat
  3. .exe буде створено в dist/

Див. BUILD_EXE.md для деталей


🎯 НАСТУПНІ КРОКИ:

1. Встановіть FFmpeg:
   Linux/macOS:  brew install ffmpeg  або  apt install ffmpeg
   Windows:      https://ffmpeg.org/download.html

2. Запустіть програму:
   python ForTheDownloader.py

3. Если потрібен .exe на Windows:
   - Перемістіть проект на Windows
   - Запустіть build_exe.bat


❌ ЯКЩО ВИНИКЛИ ПРОБЛЕМИ:

"ModuleNotFoundError: No module named 'yt_dlp'"
  → pip install -r requirements.txt

"FFmpeg not found"
  → Встановіть FFmpeg на систему

"Permission denied" (Linux)
  → chmod +x run.sh ./dist/YouTube-Downloader

"Antivirus blocks .exe"
  → Додайте в виключення антивіруса
